# Snake
## 自律分散制御で動くヘビのシミュレータ
頭部の角度を周期的に変化させるだけで蛇のように進む． <br>
頭部以外の体節は，自分の角度を一個前の体節の角度に合わせようとする（前ならえ）するだけ．  <br>
運動方程式に基づいて各体節の角度および座標・速度を更新していく．
